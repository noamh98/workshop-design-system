# SKILL: Project Run Instructions

Workshop Design System — pure HTML/CSS/vanilla JS, no build step, no server required.

- **Showcase / docs**: open `index.html` directly in a browser (double-click, or `start index.html` on Windows).
- **Slide templates**: open any file in `templates/` directly — each is a self-contained `1920x1080` slide.
- Everything works over `file://`; icons are inlined per-page (see `docs/README.md` "Icons" section) precisely so no server is needed.
- To preview with a local server instead (optional, e.g. for testing over `http://`): `npx serve .` or `python -m http.server` from the project root.
- No `npm install`, no `dotnet build` — there is nothing to compile.
