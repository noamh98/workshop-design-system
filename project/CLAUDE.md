# הוראות קבועות

- המשתמש מבקש לעיתים "המשך אוטומטית בשעה X" — פירושו: כשמגיעה הודעה חדשה (חידוש מכסה), המשך מיד את המשימה הפתוחה ברשימת המשימות בלי לשאול שאלות.
- משימה פתוחה (AUTO-TASK, עודכן 11.7): הרחבת המערכת — ראה todos. עם כל הודעה חדשה (גם ריקה/חידוש מכסה) המשך מיד, ללא שאלות, לפי הסדר: (1) js/icons.js ~20 אייקונים חדשים באותו סגנון (24×24, קו 1.75, round, currentColor); (2) js/charts.js סוגים חדשים: bar-h RTL, stacked-bar, scatter, gantt, waterfall + rough לכל הסוגים; (3) js/sketch.js strokes חדשים: bracket, underline-double, underline-wavy, scribble, star, arrow-curve, loop, strike-diag; (4) עדכון guidelines/icons.card.html, sketch-strokes.card.html, sketch-charts.card.html + check_design_system.
- שפה: עברית, RTL. סגנון: לפי מערכת העיצוב הקיימת (styles.css, js/icons.js, תבנית deck.css של המצגות הקיימות ב-templates/).
